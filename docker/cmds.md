# Docker Commands

Cheat Sheet
```
docker run <image_name>              // creates container from image and starts it
-d                                   // deatched, run in background
-p <container-port>:<host-port>      // publish container port to docker host
--name                               // assign name to container
-t                                   // create a pseudo-terminal for container
-i                                   // pass STDIN to pseudo-terminal (created with -t)
```
